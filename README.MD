## 一些声明
本扩展是为了对[https://sm.ms/](https://sm.ms/)的功能做封装而开发的。
主体代码基于[WeiboPicBed](https://github.com/Suxiaogang/WeiboPicBed)

## Features
- 支持点选/拖拽2种方式上传图片至https://sm.ms/图床  
- 可生成图片链接,HTML,UBB和Markdown四种格式
- 支持返回https安全协议的图片地址
- 支持自定义Chrome菜单栏图标
- 上传历史浏览和删除
